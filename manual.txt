ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ БАН-БОТА

1. НАСТРОЙКА БОТА
----------------
1.1. Получите токен бота у @BotFather в Telegram
1.2. Вставьте полученный токен в переменную BOT_TOKEN в файле ban_bot.py
1.3. Добавьте бота администратором в ваш канал

2. УСТАНОВКА ЗАВИСИМОСТЕЙ
------------------------
2.1. Откройте командную строку
2.2. Перейдите в папку с ботом
2.3. Выполните команду: python dependencies.py
2.4. Дождитесь сообщения "Зависимости успешно установлены."

3. ЗАПУСК БОТА
-------------
3.1. Откройте командную строку
3.2. Перейдите в папку с ботом
3.3. Выполните команду: python ban_bot.py
3.4. Дождитесь сообщения "Бот запущен"

4. ИСПОЛЬЗОВАНИЕ
--------------
4.1. Установите юзернейм канала:
    /setchannel <юзернейм канала>
    Пример: /setchannel @mychannel

4.2. Команда для бана пользователя:
    /ban <ID пользователя>
    Пример: /ban 123456789

4.3. Команда для разбанивания пользователя:
    /unban <ID пользователя>
    Пример: /unban 123456789

4.4. Как получить ID пользователя:
    - Переслать сообщение пользователя боту @getmyid_bot
    - Использовать бота @username_to_id_bot

5. ОСТАНОВКА БОТА
---------------
5.1. Нажмите Ctrl+C в командной строке
5.2. Дождитесь сообщения "Бот остановлен"

ПРИМЕЧАНИЯ:
- Бот должен быть администратором канала
- Для бана и разбанивания пользователя вам нужны права администратора
- В случае ошибки бот сообщит о причине
- Бот проверяет, есть ли у вас права администратора для выполнения команд
